Questão 1)O que é o git?Como você definiria o github?
O git é um mecanismo que possibilita a contribuição de diferentes usuários para com seu projeto/código. O github é um espaço de interação entre usuários com a finalidade de remeter projetos ou códigos. O mesmo adota esse mecanismo (git).   


Questão 2) O que você pôde notar de diferente? Comente.
A possibilidade da criação de diretórios, tornando o github um espaço em comum como, no caso, a sala de aula.
