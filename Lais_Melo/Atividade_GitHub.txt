Questão 01 - O que é o git? Como você definiria o github?
O git é um sistema de controle de versão de arquivos, onde se pode desenvolver projetos juntamente com outras diversas pessoas. 
O github é uma plataforma que permite que o usuário hospede seus projetos. É muito semelhante a uma rede social, pois permite que o usuário siga projeto de outros desenvolvedores.
